# SysAdm Notes

Кроссплатформенное (Windows & Android) оффлайн-приложение для системных администраторов и сетевых инженеров. Личный справочник команд и настроек оборудования.

## Возможности

- Иерархическая база: Вендор ➔ Технология ➔ Карточка.
- Блочная структура карточки (Текст, Код с подсветкой, Скриншоты).
- Мгновенный глобальный поиск с подсветкой результатов.
- Умный Импорт/Экспорт (JSON/ZIP) с поддержкой изображений и защитой от дубликатов.
- Полностью оффлайн работа (SQLite).

## Сборка

Для сборки из исходного кода:

- Установите Flutter SDK.
- Выполните flutter pub get.
- Запустите flutter build windows или flutter build apk
